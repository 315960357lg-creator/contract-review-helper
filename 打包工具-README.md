# 🎉 Windows打包工具已准备完毕！

## 快速开始（3步）

在Windows电脑上：

### 1️⃣ 准备Python
```
下载并安装 Python 3.8+
勾选 "Add Python to PATH"
```

### 2️⃣ 双击运行
```
一键打包.bat
```

### 3️⃣ 选择打包方式
```
选择 2 - 完整打包（推荐）
```

**完成！** 📦 输出文件在 `release/` 文件夹

## 🎯 分发方案

### 方案1：单文件 ⭐ 最简单
```
文件: dist/合同审查小助手.exe
大小: ~150-200MB
用户: 双击即用
```

### 方案2：便携版 ⭐ 推荐
```
位置: release/便携版/
包含: exe + 说明文档
用户: 解压后运行
```

### 方案3：安装程序 ⭐ 最专业
```
文件: 合同审查小助手_安装程序.exe
特点: 安装向导 + 快捷方式
用户: 运行安装程序
```

## 🚀 GitHub自动构建（可选）

如果使用GitHub托管代码：

1. 推送代码到GitHub
2. 创建版本标签：
   ```bash
   git tag v1.0.0
   git push origin v1.0.0
   ```
3. GitHub自动构建32位和64位版本
4. 在Releases页面下载

配置文件：`.github/workflows/build-windows.yml`
