# ✅ GroupBox标题遮挡问题修复

## 📋 问题描述

"审查配置"、"合同原文"、"与AI律师对话"等GroupBox的标题被顶部遮挡，无法完整显示。

## 🔍 问题原因

QGroupBox的CSS样式中：
- `margin-top: 10px` - 太小的上边距
- `padding-top: 10px` - 太小的内边距
- 导致标题文字被边框截断

## ✅ 修复方案

### 修复前
```css
QGroupBox {
    margin-top: 10px;
    padding-top: 10px;
}
QGroupBox::title {
    padding: 0 5px;
}
```

### 修复后
```css
QGroupBox {
    margin-top: 15px;      /* 增加到15px */
    padding-top: 20px;    /* 增加到20px */
    background-color: white;
    color: #333;
}
QGroupBox::title {
    padding: 0 5px 0 5px;  /* 添加左侧padding */
    color: #333;
}
```
