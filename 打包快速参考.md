# 合同审查小助手 - 打包快速参考

## 🎯 三种打包方式

### 方式1：Windows本地打包（最推荐）⭐
```batch
# 在Windows电脑上，直接双击运行
build_windows.bat
```
**输出**: `dist/合同审查小助手.exe` (150-200MB)

### 方式2：完整安装包
```batch
# 运行完整打包流程
installer\build_all.bat
```
**输出**:
- `合同审查小助手_安装程序.exe` - 标准安装程序
- `release/便携版/` - 便携版文件夹

### 方式3：GitHub自动构建（最省事）🚀
```bash
# 推送版本标签，自动构建并发布
git tag v1.0.0
git push origin v1.0.0
```
**输出**: 在GitHub Releases页面自动下载

## 📦 分发给用户

### 推荐：单文件分发

发送文件: 合同审查小助手.exe

### 专业：安装程序分发
